# br*AI*n
A machine learning libary, made completely from scratch.

### Planned features
- General
    - python wrapper
    - c equivalent of python implimentation
    - loading, and automatic saving, of trained models
- Activation & loss
    - Softmax
    - Sigmoid Activation
    - ReLU Activation
    - Categorical Cross Entropy loss
- Models
    - ConvNet
    - UNets
    - Diffusion Model
    - Transformer Language model
